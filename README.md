# Trip Ranking Simulator

Trip Ranking Simulator is a Nuxt.js application that ranks travel trips based on multiple criteria including style preferences, financial considerations, seasonal timing, and diversity in destinations. It calculates scores for each trip and sorts them to provide a ranked list for users to decide on their next adventure.

## What I Tried Out

- Setting up a Nuxt project with TypeScript
- Creating composables for reusable ranking logic
- Organizing data in separate files
- Implementing scoring utilities
- Building a Vue component with a table display
- Using Nuxt auto-imports for composables
